# Data Table Candidate Assessment

This project is a small Angular application built for the frontend assessment.

It contains a reusable data table component built from scratch with:

- Angular
- TypeScript
- plain HTML
- hand-written SCSS
- no UI libraries

The table supports:

- search across visible columns
- column sorting
- pagination
- configurable rows per page
- loading state
- empty state
- custom cell rendering, used for the status badge

## Install

```bash
npm install
```

## Run the Angular app

```bash
npm start
```

Then open:

```txt
http://localhost:4200
```

## Run Storybook

```bash
npm run storybook
```

Then open:

```txt
http://localhost:6006
```

## Build the project

```bash
npm run build
```

## Build Storybook

```bash
npm run build-storybook
```
